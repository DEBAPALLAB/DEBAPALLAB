## 🚀 Featured Projects

### 🧠 [Luman Workspace](https://luman-five.vercel.app) — `Next.js 14 · Supabase · OpenAI · Vercel AI SDK · Tiptap`
> Multi-tenant AI-powered internal workspace — Notion-style editor, task board, and calendar unified into a team OS.
- Role-based permissions (Founder, Intern, etc.) with org-wide invite flows and enterprise-grade multi-tenancy via Supabase Auth
- In-editor AI via Vercel AI SDK + OpenAI; editor built on Novel + Tiptap with LaTeX, code highlighting, and rich media
- Infrastructure: Vercel Analytics, Blob, KV Store, Speed Insights; monorepo with Turbo + pnpm

### 🍓 [Strawberry](https://strawberry-v1.vercel.app) — `Next.js · LLM Agents · GSS Data · Graph Simulation`
> Decision intelligence platform simulating how strategic narratives cascade through populations.
- Trinity Engine: behavioral profile synthesis + LLM-powered agent reasoning + social graph simulation (weak-tie theory, small-world networks)
- Grounded in GSS 2024 sociometric data — 1,499 behavioral vectors
- Stress-test strategies, identify adoption bottlenecks, predict non-linear tipping points

### 💭 [Not A Prompt](https://notaprompt.in) — `Next.js · React · OpenAI API · Tailwind CSS`
> Pre-AI thinking layer — structured decision breakdown before you touch a prompt.
- Question-generation engine and state machine for intent clarification
- Platform-aware prompt compilation for Claude, ChatGPT, and Cursor

### 📊 [CompareFi](https://comparefi.in) — `React · Tailwind CSS · REST APIs`
> Financial instrument comparison engine — clean, data-heavy UI optimized for performance and mobile.
